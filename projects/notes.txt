
The projects directory needs an info.json, this specifies every
project to be considered.

    projects: [string]

Each project must have an info.json file at the top
level of its directory.

info.json file must have the following keys:

    title: string
    subtitle: string
    month: number (1-12)
    year: number 
    links: [{name:string, url:string}]
    skills: [string]
    body: [
        {
            contentType: ("fg" | "pg")
            
            // if "fg"
            relpath: string,
            description: string

            // if "pg"
            lines: [string]
        }
    ]
    icon: string

NOTE: paths for icon and figures should be
relative to the project subdirectory.
