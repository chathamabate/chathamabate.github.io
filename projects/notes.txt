
The projects directory needs an info.json, this specifies every
project to be considered.

    projects: [string]

Each project must have an info.json file at the top
level of its directory.

info.json file must have the following keys:

    title: string
    subtitle: string
    month: number (1-12)
    year: number 
    links: [{name:string, url:string}]
    skills: [string]
    description: [string]

Each project must also have an imgs subdirectory.
